## Installation & Usage

### Using npx (Recommended)
```bash
npx cli-gf
```

### Installing Globally
```bash
npm install -g cli-gf
cli-gf
```

### Running from Source
```bash
git clone https://github.com/1dev-hridoy/cli-gf.git
cd cli-gf
npm install
node index.js
```

## How to Use

1. Run the CLI using any of the methods above
2. Enter your name when prompted
3. Select an AI character to chat with
4. Start chatting! Type "exit" to quit anytime
